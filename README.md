###基本代码规范:

1.所有的类命名均以GBM打头,例如,登陆页面写为GBMLoginViewController

2.所有的控件命名都要写明是什么控件,如:登陆按钮loginButton,密码输入框passWordTextFiel,就是要做到见名知意,还要见名知类型.

3.方法名也是一样,一定要做到见名知意.

4.遵循MVC,所有的UI层都写到view里

5.一定要写注释!

###使用worktile:

1.每个人的任务都会分配到worktile上及时查看

2.项目资源文件也上传到了worktile

3.完成任务后及时勾选掉相应的任务进度
